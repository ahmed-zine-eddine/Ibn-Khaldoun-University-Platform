# Design System Usage

## Tokens
- `tokens/colors.js`: Color palette for all components
- `tokens/spacing.js`: Spacing scale
- `tokens/typography.js`: Font families, sizes, weights

## Components
- `components/Button.jsx`: Example of a reusable, themed button

## Theming
- All components should use tokens for style and support future theming

## How to Use
- Import components from `design-system/components`
- Use tokens for custom styles
